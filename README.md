# AI Skill Gap Analyzer 🚀

This project analyzes resumes and compares them with job descriptions to identify skill gaps and suggest learning paths.

## 🔍 Features
- Resume text analysis
- Job role matching
- Skill gap identification
- Learning recommendations

## 🛠 Tech Stack
- Python
- NLP (Scikit-learn, Pandas)

## 🚧 Status
Currently under development

## 🎯 Goal
To help students understand what skills they lack and how to improve for specific job roles.
